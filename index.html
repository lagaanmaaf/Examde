📋 How to Create a Test

Complete guide to building your exam using Excel upload

Step 1 — Excel Column Format

Create an Excel file (.xlsx) with exactly these column headers in Row 1:

ColumnDescriptionExample
QUESTIONSThe full question textWhat is the capital of India?
AOption A textMumbai
BOption B textNew Delhi
COption C textChennai
DOption D textKolkata
CORRECT ANSWERLetter(s) of correct option(s). For multiple correct answers, separate with comma.B or B,C
MAXIMUM MARKSMarks awarded for correct answer (optional — uses section default if blank)2
NEGATIVE MARKINGMarks deducted for wrong answer (optional — uses section default if blank)0.5
Step 2 — Single vs Multiple Correct Answers

For a question with one correct answer, put just the letter in CORRECT ANSWER:

CORRECT ANSWER → B

For a question with two or more correct answers, separate them with a comma. The student must select ALL correct options to get marks — selecting only one will count as wrong:

CORRECT ANSWER → B,C (student must pick both B and C)

⚠️ Multi-answer questions automatically allow selecting multiple options during the test. Single-answer questions lock to one choice.

Step 3 — Sample Excel Data

Your Excel should look like this (Row 1 = headers, Row 2 onwards = questions):

QUESTIONS | A | B | C | D | CORRECT ANSWER | MAXIMUM MARKS | NEGATIVE MARKING Capital of India? | Mumbai | New Delhi | Chennai | Kolkata | B | 2 | 0.5 Planets in solar… | 7 | 8 | 9 | 10 | B | 2 | 0.5 Primary colours? | Red | Blue | Green | Yellow | A,B,C | 3 | 0
